# 🎓 Maurya International University of Technology & Management (MIUTM)

A modern **Full-Stack University Management System** developed using **Flask, SQLite3, HTML5, CSS3, Bootstrap, and JavaScript**. This project simulates a professional university website with admission management, academic information, student interaction, and administrative functionalities.

The application is designed with a responsive user interface and deployed on **Render**, making it accessible from anywhere.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Nitin765kumar/MIUTM_PROJECT.git
```

Move into project directory

```bash
cd MIUTM_PROJECT
``

Run Application

```bash
python app.py
```

Open browser

```
http://127.0.0.1:5000
```

---

# 🚀 Deployment

This application is successfully deployed on **Render**.

Deployment Platform:

- Render

Build Command

```bash
pip install -r requirements.txt
```

Start Command

```bash
gunicorn app:app
```
